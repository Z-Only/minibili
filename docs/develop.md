# 开发

## Getting Started

1. [Tauri 前置依赖](https://tauri.app/zh-cn/start/prerequisites/)
2. 配置 node 环境
3. 安装依赖，推荐 pnpm
4. 本地启动

```shell
pnpm install
pnpm tauri dev
```
